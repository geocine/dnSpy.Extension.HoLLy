Hello,

When analyzing files using dnSpy, the renaming feature comes in handy. However, renaming using the Edit member dialog isn't very "smart". It only renames the current member and doesn't take into account that the name which might be important for inheritance. I suggest a new option in the context menu for members for "Smart renaming" which would rename the current member as well as any other members that may need renaming because of inheritance.

Thanks in advance!
